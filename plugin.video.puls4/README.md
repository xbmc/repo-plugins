Puls4 XBMC Addon
=======
Puls4 is an addon that gives you access to the Puls4 Video Platform.


Requirements
------------
Tested on Jarvis


Supported platforms
-------------------
Windows, Linux , Android and OSX


Current Features
----------------
* All Shows
* H264 Stream (HD/SD)
* Highlights
* Missed a Show?
* New Videos
* Popular Videos


Known Issues
------------
* None so far
